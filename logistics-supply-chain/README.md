# 🚛 Logistics & Supply Chain

Agentic use cases for shipping and logistics operations. Companion classic-ML versions
of several of these problems live in
[Logistics_UseCases](https://github.com/immu4989/Logistics_UseCases); the entries here
are the agentic formulation — an LLM agent that investigates with tools and commits to
a decision, verified per [VERIFICATION.md](../VERIFICATION.md).
